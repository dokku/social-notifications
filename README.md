# stackoverflow-notifications

Sends notifications to slack whenever a new question with the `dokku` tag is posted.

## Config

- DATABASE_FILE
- LITESTREAM_ACCESS_KEY_ID
- LITESTREAM_REPLICA_URL
- LITESTREAM_SECRET_ACCESS_KEY
- LOG_FORMAT
- SLACK_CHANNEL
- SLACK_ICON_URL
- SLACK_USERNAME
- SLACK_WEBHOOK_URL
- TAG
